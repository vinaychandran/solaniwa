## 構成
[src]ディレクトリが開発用です。
ビルドのコマンドで[dist]ディレクトリに自動生成されます。
必要なパッケージは[node_modules]ディレクトリにインストールされます。

## コマンド
下記のコマンドでパッケージをインストールしてください。
```
# インストール
$ npm install

# 開発
$ npm run start

# ビルド
$ npm run build
```

## HTML
テンプレートエンジンに[EJS]を使用しています。

[Sass]をScss記法で使用しています。
